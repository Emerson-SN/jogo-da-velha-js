#  Jogo da Velha em JavaScript

Jogo da velha simples desenvolvido com **HTML e JavaScript puro**, com foco na prática de lógica de programação e manipulação do DOM.

##  Funcionalidades
- Dois jogadores (X e O)
- Alternância automática de turnos
- Verificação de vitória
## 🛠 Tecnologias Utilizadas
- HTML5
- JavaScript (Vanilla JS)
